## 🧩 Topic 15 — Function Scope & Return Values

I created a new practice exercise for Topic 15.

The exercise helped me understand how a function can:

1. Receive information through parameters.
2. Process that information.
3. Return a result.
4. Pass that result to another function.
5. Use the final result in the main program.

One of the most important ideas I practiced today was that functions do not have to work independently.

A function can return a value that another function can use.

This helped me better understand how larger programs can be divided into smaller and more manageable pieces.
